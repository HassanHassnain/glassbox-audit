# BlackboxNLP 2026 submission source

Anonymized ACL-format submission for the **regular archival track** of
BlackboxNLP 2026 (co-located with EMNLP 2026). The regular track fits
Glassbox as original unpublished work; the special track is scoped to
reproducing established results. Official limits: **8 pages + unlimited
references and appendices**, with Limitations/Ethics permitted on one
additional page. Direct-submission deadline: **July 17, 2026 (AoE)**.

## Contents

| File | Purpose |
|---|---|
| `main.tex` | the paper (review mode: line numbers, anonymized) |
| `references.bib` | verified bibliography (venues confirmed against the ACL Anthology / proceedings / arXiv; unconfirmed venues cited as arXiv preprints) |
| `acl.sty`, `acl_natbib.bst` | official ACL style files, vendored verbatim from [acl-org/acl-style-files](https://github.com/acl-org/acl-style-files) |
| `figures/` | submission copies of the two hardening figures (canonical source: `docs/figures/*.pdf`, regenerated by `scripts/generate_hardening_figures.py`) |

## Build

```bash
tectonic main.tex          # one command; runs BibTeX automatically
# or classic:
pdflatex main && bibtex main && pdflatex main && pdflatex main
```

The build is clean (no overfull boxes); the compiled draft is 12 pages:
8 pages of content, Limitations/Ethics on the permitted extra page,
then references and appendices A–H.

## Provenance rules

- Every number in `main.tex` is transcribed from a committed evidence file
  under `results/` (see the claim-to-evidence map in Appendix H).
- The repository link is withheld in the PDF for review-time anonymity;
  restore it (and switch `\usepackage[review]{acl}` to `\usepackage{acl}`)
  for the camera-ready.
- If figures are regenerated, re-copy the PDFs from `docs/figures/` into
  `figures/` so the submission stays in sync with the evidence.
