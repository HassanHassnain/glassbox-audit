# BlackboxNLP 2026 submission source

This is an anonymized ACL-format draft for the regular archival track. It uses the official ACL style (`acl.sty` and `acl_natbib.bst`) supplied by the current ACL/EMNLP template; those template-owned files are intentionally not copied into this repository.

The regular track is the appropriate fit for Glassbox as original unpublished work. The special/reproducibility track is scoped to reproducing established results. The official call lists an 8-page limit plus references for full papers and a July 17, 2026 direct-submission deadline.

Compile from this directory after copying the current official ACL style files here, or upload `main.tex`, `references.bib`, and the two PDFs under `docs/figures/` to an official ACL Overleaf project.
