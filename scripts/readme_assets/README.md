# README assets

Everything in `docs/assets/` is generated by the scripts in this directory, so the
visual identity stays reproducible — same palette, fonts, and panel language across
the hero, diagrams, result figures, and the demo GIF.

```bash
pip install pillow          # only dependency
python scripts/readme_assets/generate_all.py
```

| Script | Output |
|---|---|
| `generate_hero.py` | `docs/assets/hero.png` |
| `generate_pipeline.py` | `docs/assets/pipeline.png` |
| `generate_causal.py` | `docs/assets/fig_causal_effects.png` |
| `generate_stability.py` | `docs/assets/fig_stability_grid.png` |
| `generate_component.py` | `docs/assets/fig_component_path.png` |
| `generate_ladder.py` | `docs/assets/fig_claim_ladder.png` |
| `generate_dose.py` | `docs/assets/fig_dose_response.png` |
| `generate_demo_gif.py` | `docs/assets/demo.gif` |

`style.py` holds the shared identity (palette, fonts, glass panels, chips). Figure
values are transcribed from the committed evidence summaries under `results/` —
each figure's footer names its source file. If a result JSON changes, update the
matching constants in the figure script.

Fonts (`fonts/`): JetBrains Mono and Space Grotesk, both under the SIL Open Font
License (see the bundled OFL files).
